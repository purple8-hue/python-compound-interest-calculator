# python-compound-interest-calculator
## Compound Interest Calculator

A command-line tool that calculates compound interest based on user-provided principal, interest rate, and time period, with input validation to prevent negative values.

### How it works
- Prompts for principal amount, interest rate, and time in years
- Validates that none of the inputs are negative, re-prompting if necessary
- Calculates and displays the final balance using the compound interest formula
